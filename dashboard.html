securityHTTPS / "Not Secure" Fix
Why it shows "Not Secure": Your server is running HTTP, not HTTPS. Browsers warn about all HTTP sites.

Fix option 1 — Render.com (recommended): Deploy to Render.com — it auto-assigns a free *.onrender.com domain with TLS included. No configuration needed.

Fix option 2 — Nginx reverse proxy: Put Nginx in front of Flask with a Let's Encrypt certificate. Use Certbot to get a free cert.

Fix option 3 — Cloudflare Tunnel: Use cloudflared tunnel to expose your local server over HTTPS for free.

Note: HTTPS is required for WebSocket (wss://) to work reliably in browsers.
